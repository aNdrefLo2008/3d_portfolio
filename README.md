# 3D Developer Portfolio

An interactive 3D developer portfolio built with React, Tailwind CSS, and Vite.  
Designed to present technical projects in a modern, performant, and visually engaging way.

## 🚀 Features

- Interactive UI components
- Lazy-loaded media for performance optimization
- Responsive design
- Internationalization (i18n support)
- Optimized production build (Vite + Vercel ready)

## 🛠 Tech Stack

- React
- Tailwind CSS
- Vite
- i18next (internationalization)
- PostCSS
- Vercel (deployment)

## 📁 Project Structure


public/ # Static assets
src/
├── assets/ # Images, media
├── components/ # Reusable React components
├── constants/ # Configuration values
├── hooks/ # Custom hooks
├── utils/ # Utility functions
├── App.jsx # Root component
└── main.jsx # Entry point


## ⚙️ Performance Optimizations

- Lazy loading for video components
- Optimized asset delivery
- Modular component structure
- Clean code separation

## 💡 Purpose

This portfolio serves as a centralized platform to showcase:

- Full-stack backend systems
- Data Science & Machine Learning projects
- System architecture work
- Analytical dashboards
